# 📱 智能手机选购助手

基于官方数据的多维度手机筛选工具，帮助你找到最适合的手机。

## 功能特点

- **230+ 款机型**：覆盖苹果、华为、小米、OPPO、vivo、三星等12个品牌
- **多维度筛选**：品牌、屏幕形态、处理器、特性标签（可多选叠加）
- **丰富标签**：潜望长焦、大电池、轻薄、防水、NFC、红外、USB3.0、无线充电、散热风扇等
- **多种排序**：最新发布、电池容量、重量、屏幕大小、快充功率
- **详细参数**：处理器、屏幕、电池、充电、USB版本、重量、系统、摄像头等
- **数据准确**：所有数据来自各品牌官网，截止到2026年5月6日
