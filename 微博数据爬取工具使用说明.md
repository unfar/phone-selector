# 微博数据爬取工具使用说明

## 工具概述
微博数据爬取工具用于从微博获取手机参数信息，补充官网缺失的数据，提供多数据源支持。

## 功能特性

### 1. 手机信息搜索
- 根据品牌和型号搜索微博中的手机参数
- 自动提取处理器、屏幕、电池、充电等核心参数
- 支持模糊搜索和关键词匹配

### 2. 官方微博搜索
- 搜索品牌官方微博账号
- 获取最新手机发布信息
- 提取新品参数和规格

### 3. 参数自动提取
- 智能识别微博文本中的参数信息
- 支持多种参数格式和表达方式
- 自动标准化参数名称和单位

### 4. 数据验证
- 集成参数验证器，确保数据合理性
- 自动纠正明显错误
- 标注数据来源为微博

## 使用方法

### 基本搜索

```python
from weibo_scraper import WeiboPhoneDataScraper

# 创建爬取器实例
scraper = WeiboPhoneDataScraper()

# 搜索手机信息
phone_info = scraper.search_phone_info("Pura 90", "华为")
print(phone_info)
```

### 官方微博搜索

```python
# 搜索品牌官方微博
official_infos = scraper.search_official_weibo("华为")
for info in official_infos:
    print(info)
```

### 数据验证

```python
# 验证微博数据
validated_data = scraper.validate_weibo_data(phone_info, "华为", "Pura 90")
print(validated_data)
```

## 使用场景

### 1. 补充官网缺失数据
当官网没有某款机型的详细参数时，可以从微博搜索补充。

### 2. 验证现有数据
用微博数据验证现有数据的准确性，发现不一致时进行确认。

### 3. 获取最新信息
从品牌官方微博获取最新发布机型的参数信息。

### 4. 交叉验证
多数据源交叉验证，提高数据准确性。

## 注意事项

### 1. 数据准确性
- 微博数据可能存在错误，需要谨慎使用
- 优先参考官网数据，微博作为补充
- 重要参数需要多源验证

### 2. 信息时效性
- 微博信息可能不是最新的
- 注意发布时间，避免使用过时信息
- 定期更新微博数据

### 3. 使用限制
- 遵守微博使用条款
- 控制请求频率，避免被封禁
- 不要用于商业目的

### 4. 数据标注
- 使用微博数据时必须标注来源
- 区分官网数据和微博数据
- 对不确定参数保持怀疑态度

## 错误处理

### 常见错误
1. **网络错误**：检查网络连接
2. **解析错误**：检查微博页面结构变化
3. **验证错误**：检查参数验证规则
4. **权限错误**：检查微博访问权限

## 示例代码

### 完整示例
```python
from weibo_scraper import WeiboPhoneDataScraper
import json

# 创建爬取器
scraper = WeiboPhoneDataScraper()

# 搜索手机信息
phone_info = scraper.search_phone_info("Pura 90", "华为")

# 验证数据
validated_info = scraper.validate_weibo_data(phone_info, "华为", "Pura 90")

# 保存结果
with open('weibo_phone_data.json', 'w', encoding='utf-8') as f:
    json.dump(validated_info, f, ensure_ascii=False, indent=2)

print("数据已保存")
```
